# تصميم حديث لموقع Ristorante Cappuccino

هذا المجلد يحتوي على نموذج موقع حديث وجاهز للفتح مباشرة من ملف `index.html`.

## الملفات
- `index.html`: الصفحة الرئيسية كاملة.
- `styles.css`: التصميم المتجاوب والألوان والحركات.
- `script.js`: قائمة الجوال، التمرير، وتبويبات القائمة.
- `assets/logo.svg`: شعار SVG مبدئي قابل للاستبدال.

## ما الذي تم تحسينه؟
- واجهة حديثة مناسبة للجوال.
- دعوات واضحة للحجز وفتح القائمة.
- عرض أفضل للموقعين Spandau وGatow.
- أقسام مميزة للقائمة، Catering، وEventlocation.
- SEO أساسي وبيانات منظمة LocalBusiness/Restaurant.
- تجربة أسرع لأن التصميم لا يعتمد على مكتبات خارجية.

## ملاحظات قبل النشر
1. استبدل الخلفيات/الرسومات بصور حقيقية عالية الجودة للمطعم والأطباق.
2. اربط نموذج التواصل بخدمة فعلية مثل Formspree أو CMS أو WordPress plugin.
3. حدّث الأسعار والقائمة من المصدر الرسمي قبل النشر.
4. راجع نصوص Impressum وDatenschutz مع صاحب الموقع قبل الإطلاق.
